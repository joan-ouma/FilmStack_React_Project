# FilmStack

A modern React application for discovering movies, managing your personal watchlist, and exploring film details. 

## Features

- **Movie Discovery**: Browse popular, upcoming, and top-rated movies.
- **Personal Watchlist**: Add movies to your watchlist and save them for later. The watchlist persists locally in your browser.
- **Responsive Design**: Enjoy a seamless experience across desktop and mobile devices.
- **Fast Navigation**: Powered by React Router DOM for instant, client-side page transitions.

## Tech Stack

- **Frontend Framework**: React (via Vite)
- **Routing**: React Router DOM
- **State Management**: React Context API
- **Styling**: Vanilla CSS

## Project Structure

- `src/components/`: Reusable UI components (buttons, movie cards, navigation).
- `src/contexts/`: Global state management contexts (e.g., WatchlistContext).
- `src/pages/`: Main application screens (Home, Movie Details, Watchlist).
- `src/assets/`: Static assets like images and icons.

## Getting Started

### Prerequisites

Ensure you have Node.js installed (v18 or higher recommended).

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd FilmStack_React_Project/filmstack
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
